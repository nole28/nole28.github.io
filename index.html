● In active development

Espres App

Espres is a SwiftUI social app with real accounts and uploaded content. It has the main parts I wanted to learn by building: feed, image posts, text posts, video upload, short-form playback, comments, likes, profiles, user search, settings, account actions, and view counts.

The app uses Firebase Authentication, Firestore, and Firebase Storage. Most of the work was not just UI — I also handled upload states, loading states, navigation bugs, validation, user/profile data, media files, and issues that only showed up when running the app.

Authentication

Users can create accounts, log in, update account data, and use password-related account flows through Firebase.

Feeds and posts

The app supports image posts, text posts, comments, likes, and profile-linked content from real users.

Video upload

Videos are picked from the device, prepared for upload, stored in Firebase Storage, and displayed back inside the app.

Short-form playback

Espres includes a reels-style video player with playback, interaction buttons, and view tracking.

Profiles and search

Users can view their own profile, open other user profiles, and search through users and content.

Settings

The settings flow includes account management, profile state, and sensitive actions like password updates.
