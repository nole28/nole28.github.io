Espres

Main project: a SwiftUI social app.

Espres is the app I use to show what I can build right now. It includes account flow, feed screens, videos, upload UI, profile screens, saved posts, settings, and messaging.

What it is

A social media-style iOS app built with SwiftUI and Firebase. The goal was to build more than static screens: navigation, user accounts, media screens, app state, Firestore data, and flows that connect together.

Stack

Swift SwiftUI Firebase Auth Firestore Firebase Storage AVKit PhotosUI Git
Features

What I built in the app.

The main parts I worked on while building Espres.

Auth

Login, sign up, session state, password screen, settings, sign out, and delete account UI.

Feed

Photo posts, text posts, segmented feed, post actions, comments entry, and saved posts.

Video

Video list, short-style video screen, video detail UI, saves, comments, and view count display.

Upload

Gallery picker flow, title and description inputs, privacy options, and upload screen structure.

Profiles

Profile grid, edit profile, saved posts, following count, and public profile screens.

Messages

Messages list, chat screen, message input, chat model, and conversation UI.
