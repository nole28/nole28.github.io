// 03 — Featured Project

Espres.

A full SwiftUI social video app with real authentication, Firestore data, Firebase Storage media upload, profile flows, search, comments, and account management.

● In active development

A product-style iOS app, built end to end.

Espres is my main iOS project: a SwiftUI social platform with image posts, text posts, video upload, short-form playback, comments, likes, profiles, user discovery, settings, and account flows. It uses Firebase Authentication, Firestore, and Firebase Storage, so I can practice real data flow instead of only building static UI.

The goal was to build something closer to a real product than a small demo: multiple tabs, reusable views, asynchronous upload state, media handling, profile data, and runtime bugs that had to be solved in the simulator, not only in previews.
